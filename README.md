# Automated Multi-Environment Deployment Pipeline

## Overview

This project demonstrates a production-style DevOps pipeline that automates application deployment across three Kubernetes environments:

- Development
- Staging
- Production

The project integrates Infrastructure as Code (Terraform), Configuration Management (Ansible), Continuous Integration and Continuous Delivery (Jenkins), Containerization (Docker), Kubernetes orchestration, Monitoring, Logging, and Security best practices.
